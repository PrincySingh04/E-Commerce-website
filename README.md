# 🛒 Amazon Clone Website

A fully responsive and interactive Amazon Clone Web Application built using **HTML5, CSS3, bootstrap and JavaScript**. This project replicates the core shopping experience of Amazon with a modern user interface, dynamic cart functionality, login popup system, product pages, and responsive design.

The goal of this project was to strengthen frontend development skills by implementing real-world e-commerce features using  JavaScript and responsive web design techniques.

---

# 🚀 Features

## 🔐 Authentication Interface

* Login Popup System
* Interactive Login Modal
* Smooth Popup Animations

## 🛍️ Shopping Features

* Add to Cart Functionality
* Dynamic Cart Popup
* Cart Quantity Management
* Real-Time Total Price Calculation
* Buy Now / Order Confirmation Popup

## 🔎 Navigation & Search

* Search Bar User Interface
* Category-Based Navigation
* Multiple Product Pages
* Easy Product Browsing Experience

## 🎨 User Interface & Design

* Fully Responsive Layout
* Hero Image Slider
* Modern Amazon-Inspired Design
* Smooth Hover Effects
* Animated Product Cards
* Back To Top Button

## 📦 Product Showcase

* Multiple Product Sections
* Product Cards with Images and Pricing
* Dedicated Product Pages (Part2, Part3, Part4, etc.)
* Interactive Shopping Experience
