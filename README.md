🛒 Amazon Clone Website

A fully responsive and interactive Amazon Clone Web Application built using HTML, CSS, and JavaScript.
This project includes a modern UI design, product sections, cart functionality, login popup, animated cards, product pages, and order confirmation popup.
🚀 Features
1.Login Popup System
2. Add to Cart Functionality
3. Dynamic Cart Popup
4. Order Confirmation / Buy Popup
5. Search Bar UI
6. Multiple Category Navigation
7. Smooth Hover & Animation Effects
8. Responsive Layout
9. Hero Image Slider
10. Product Pages (Part2, Part3... etc.)
11. Cart Quantity & Total Price Update
12. Back To Top Button


🛠️ Technologies Used
HTML5
CSS3
JavaScript
Bootstrap Classes
